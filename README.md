## 🧠 Projects Highlighted

### 1. Crop Classification Using Sentinel‑1 SAR and Machine Learning

- Built a crop classification pipeline using Sentinel‑1 SAR time-series data.
- Implemented a **Random Forest** classifier to generate crop-type maps.
- Demonstrated strong performance with high overall accuracy and solid class-wise metrics.
- Integrated into the portfolio via a **Satellite Map Explorer** that lets users:
  - Toggle simulated SAR and crop layers.
  - Inspect crop type and confidence at different locations.
  - Simulate predictions by clicking on the map.

### 2. House Price Prediction Model

- Implemented a regression-based ML model to estimate house prices.
- Used **Python**, **Pandas**, **NumPy**, and **Scikit-learn** for:
  - Data cleaning and preprocessing.
  - Feature engineering and selection.
  - Training, evaluating, and iterating over models.
- Presented as a compact secondary project card in the portfolio.

## 🚀 Getting Started

You can run this project locally with just a browser.

### Prerequisites

- Any modern web browser (Chrome, Firefox, Edge, etc.)
- Git (optional but recommended)

### Clone the repository

```bash
git clone https://github.com/swati30kumari05/swati-portfolio.git
cd swati-portfolio
```

### Run locally

Just open `index.html` in your browser:

- Double-click `index.html`, or
- Use a simple local server (recommended):

```bash
# Python 3
python -m http.server 8000
# Then open http://localhost:8000 in your browser
```

The Leaflet-based **Satellite Map Explorer** will load its tiles and overlays as long as you are online.
